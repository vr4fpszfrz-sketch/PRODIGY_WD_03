# PRODIGY_WD_03 — Tic-Tac-Toe Web Application

## Task
Build an interactive Tic-Tac-Toe web app using HTML, CSS, and JavaScript. Handle user clicks, track game state, and check win conditions. Support play against another player or an AI opponent.

## Tech Used
- HTML5
- CSS3 (Grid layout)
- Vanilla JavaScript

## Features
- 2-player mode and vs-AI mode (toggle button)
- Simple AI: tries to win, blocks the player, otherwise picks center/corner/random
- Win detection across all 8 winning patterns, with highlighted winning cells
- Draw detection
- Restart button to reset the board instantly

## How to Run
1. Clone this repo
2. Open `index.html` in any browser
